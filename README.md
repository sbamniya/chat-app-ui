# Chat App UI

Simple React + Vite + TypeScript with, `graphql-request`, `@tanstack/react-query`, `antd` and `socket.io`.

Application uses `pnpm` to install dependencies.

## Start Development Server

### Install dependencies

```sh
pnpm install
```

or

```sh
pnpm i
```

### Run server

```sh
pnpm start
```

### Build production

```sh
pnpm run build
```
